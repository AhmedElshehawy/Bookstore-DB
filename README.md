# Bookstore-DB

## Description

This is a simple API for managing books in a database. It is built with FastAPI and uses Mangum for AWS Lambda compatibility.

## Requirements

- Python 3.10+
- FastAPI 0.115.8
- Mangum 0.19.0
- Pydantic 2.10.6
- Pydantic Settings 2.7.1
- Psycopg2 2.9.10
- Python-Dotenv 1.0.1
